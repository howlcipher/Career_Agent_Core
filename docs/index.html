Core Architecture

Engineered for absolute stealth, resilience, and efficiency.

  • Self-Healing DOM Invalidates stale Playwright CSS mappings if a website updates its UI, forcing a structural re-learn.
  • Dynamic Learner Module Autonomously reads unknown Applicant Tracking Systems (e.g. Workday), sends the form to Gemini to mathematically map input selectors, caches the blueprint, and applies.
  • Resilient Networking & Token Optimizations Aggressively prunes DOM payloads to under 1.5k characters to protect API quotas. Implements strict context timeouts to prevent networking hangs and lockups.
  • Playwright Fallback Scraper Bypasses API limits by deploying an undetectable headless DuckDuckGo/Yahoo search scraper.
  • Strict Promptsec Quarantine Secure text-parsing layer neutralizes malicious indirect prompt injections embedded in job descriptions.
  • Stealth Writer Bypasses AI detection with humanizing delays, active voice, and zero corporate buzzwords.
