Zen Architecture

Career Agent Core operates on a state-check-pointed SQLite backend. It's built for efficiency and privacy.

  • Dynamic Source Discovery: LLM pipeline scouts Himalayas, Remotive, and other untracked job boards.
  • Template Matching Loop: Identifies Greenhouse, Lever, Workday, and Taleo DOM footprints dynamically.
  • Promptsec Quarantine: Secure text-parsing layer neutralizes malicious indirect prompt injections.
  • Local PII Isolation: Your name, phone, and address remain entirely local and git-ignored.

Getting Started

# 1. Clone the repository
git clone https://github.com/howlcipher/Career_Agent_Core.git
cd Career_Agent_Core

# 2. Setup your isolated PII
cp pii.yaml.template pii.yaml

# 3. Export Gemini Auth
export GEMINI_API_KEY="your_api_key_here"

# 4. Automate
go run cmd/agent/main.go
