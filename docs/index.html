Core Architecture

Built for stealth and efficiency.

  • Self-Healing DOM: Invalidates stale Playwright CSS mappings if a website updates its UI, forcing a re-learn.
  • Playwright Fallback Scraper: Bypasses SerpApi limits by deploying an undetectable headless DuckDuckGo scraper.
  • Strict Promptsec Quarantine: Secure text-parsing layer neutralizes malicious indirect prompt injections in job descriptions.
  • Stealth Writer: Bypasses AI detection with humanizing delays, active voice, and zero corporate buzzwords.
